# Patient_Health_Management_System
Integrated Patient Health System for Biomedical Signal and Image Processing

The functionalities of this system are:
- Extract clinically relevant information for easy visualization
- Real-time management of patient records
- Automates tasks, such as noise removal from signals, image segmentation, and the initial analysis of vast datasets.
